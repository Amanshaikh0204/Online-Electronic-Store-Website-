# Online-Electronic-Store-Website-
for running this project firstly install xampp server and after installing it go to file section then select the directory where you have downloaded your xampp. let's take it as Windows C: then click on xampp then select htdocs then create new file and name it as you wanted. then paste all the folders from here to that file.
after doing this open xampp start Apache and MySQL then go to phpMyAdmin and create one database named as userform. The database name must be userform then import this userform.sql file into it then all the tables will be generated automatically. 
After Completing all these steps go to your browser and write "localhost\filename\index.php" and then it will open homepage of this website.
